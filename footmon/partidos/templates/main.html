FootMon - Estadísticas de Fútbol

Bienvenido a FootMon, la aplicación de datos estadísticos de fútbol.

Esta aplicación forma parte de nuestro trabajo, que tiene como objetivo principal explorar el potencial de las bases de datos NoSQL, como MongoDB, y alguna técnica de procesamiento de datos, como MapReduce, que son herramientas muy útiles para analizar y extraer conocimientos relevantes de conjuntos grandes de datos.

Para esto, hemos usado conjunto de datos sobre resultados, registro de goles y mucho más. ¡Explora nuestras estadísticas y resultados!
